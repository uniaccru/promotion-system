# Система оценки и продвижения сотрудников (Promotion and Grading System)

Система управления оценкой производительности сотрудников, целями, ревью и процессами продвижения по карьерной лестнице с функционалом калибровки.

## Описание

Система предназначена для автоматизации процессов оценки сотрудников, управления целями, проведения ревью и обработки заявок на повышение. Система поддерживает процесс калибровки для справедливого сравнения кандидатов на повышение.

## Основные возможности

### Управление сотрудниками
- Регистрация и аутентификация пользователей
- Профили сотрудников с информацией о текущем грейде
- История изменений грейдов
- Управление сотрудниками (для HR)

### Управление целями
- Создание шаблонов целей
- Назначение целей сотрудникам
- Отслеживание статуса выполнения целей
- Привязка целей к заявкам на повышение

### Оценка производительности
- Проведение ревью менеджерами
- Выставление оценок и комментариев
- Номинация на повышение
- История оценок

### Заявки на повышение
- Подача заявок на повышение сотрудниками
- Прикрепление обоснований и доказательств
- Прикрепление файлов к заявкам
- Привязка целей к заявкам
- Процесс ревью заявок (HR, менеджеры)
- Статусы: pending, under_review, ready_for_calibration, in_calibration, calibration_completed, approved, rejected

### Калибровка
- Создание пакетов калибровки для определенного грейда
- Назначение оценщиков для калибровки
- Сравнение кандидатов попарно
- Автоматический расчет рейтинга кандидатов
- Минимум 3 кандидата для активации калибровки

### Алгоритм калибровки
Алгоритм калибровки основан на парных сравнениях: для каждого пакета формируется набор кандидатов и два оценщика, которые последовательно сравнивают все возможные пары и выбирают победителя; для каждого кандидата считается количество побед (`wins`) и общее число сравнений (`totalComparisons`), по которым вычисляется коэффициент `winRate = wins / totalComparisons`, а итоговый рейтинг кандидатов строится по убыванию `winRate`.

### Роли пользователей
- **Employee** - сотрудник: просмотр профиля, целей, ревью, подача заявок на повышение
- **Team Lead** - тимлид: оценка сотрудников, участие в калибровке, просмотр заявок
- **HR** - HR-специалист: управление сотрудниками, ревью заявок, создание калибровок

## Технологический стек

### Backend
Java 17, Spring Boot 3.2.0, Spring Security, Spring Data JPA, PostgreSQL 15, JWT (jjwt 0.12.3), SpringDoc OpenAPI, Lombok, Gradle

### Frontend
React 18, TypeScript, Vite, Material-UI (MUI) 5, React Router 6, Axios, date-fns

### Инфраструктура
Docker, Docker Compose, Nginx, PostgreSQL

Основные таблицы:
- `users` - пользователи системы
- `employees` - сотрудники
- `grades` - грейды (Junior, Middle, Senior, Lead)
- `goals` - шаблоны целей
- `goal_assignments` - назначенные цели
- `manager_evaluations` - оценки менеджеров
- `promotion_requests` - заявки на повышение
- `promotion_request_reviewers` - ревьюеры заявок
- `promotion_request_files` - файлы заявок
- `promotion_request_goals` - связи заявок и целей
- `calibrations` - калибровки
- `calibration_evaluators` - оценщики калибровки
- `comparisons` - сравнения кандидатов
- `grade_history` - история изменений грейдов
